# pFLEX

pFLEX is a Python toolkit for evaluating CRISPR screen and co-essentiality networks against biological functional standards such as CORUM protein complexes, Gene Ontology Biological Processes, and pathways.

Use pFLEX to:

- load one or more gene-by-sample matrices
- rank gene pairs by correlation or a precomputed score
- evaluate global precision-recall performance
- evaluate performance at the functional-standard term level
- plot benchmark summaries and export result tables
